# JamaicanFlowerSeeds — Telegram-бот интернет-магазина

Telegram-бот для каталога товаров и приёма заказов семян.

## Функционал

- **Каталог** — 2 категории (Femenized, Auto Femenized), по 5 товаров с вариантами 5/10/20 семян
- **Корзина** — добавление, удаление, изменение количества
- **Оформление заказа** — пошаговый сбор ФИО, телефона, города, доставки, адреса, комментария
- **Мои заказы** — последние 5 заказов со статусами
- **Контакты / Помощь**
- **Админ-панель** — просмотр заказов, смена статусов, настройки (счёт, ссылка на менеджера)
- **Уведомления** — пользователю после заказа, администратору — полный текст заказа

## Установка

```bash
cd jamaican_flower_seeds
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env: BOT_TOKEN, ADMIN_IDS, ADMIN_PASSWORD
python bot.py
```

## Конфигурация (.env)

| Переменная | Описание |
|------------|----------|
| BOT_TOKEN | Токен бота от @BotFather |
| ADMIN_IDS | ID администраторов через запятую |
| ADMIN_GROUP_ID | ID группы для заказов (опционально) |
| ADMIN_PASSWORD | Пароль входа в админку |
| PAYMENT_BANK_NAME | Название банка для счёта |
| PAYMENT_ACCOUNT | Номер счёта для оплаты |
